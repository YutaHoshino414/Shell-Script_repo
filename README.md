# _-Shellnium-_  ShellScriptでSeleniumを試す✅ 


### Pre-Requisite

*install & activate ChromeDriver
```
$ chromedriver
```
starting it on port 9515: `http://localhost:9515`

*required jq command installed
```
$ brew install jq
```

## test.sh

```
sh test.sh
```
